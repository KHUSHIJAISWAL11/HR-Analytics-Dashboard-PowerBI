# HR Analytics Dashboard - Power BI

## Project Overview
This project is an interactive HR Analytics Dashboard built using Power BI.

The dashboard analyzes:
- Employee attrition
- Salary trends
- Gender distribution
- Workforce demographics

## Tools Used
- Power BI
- Power Query
- DAX
- CSV Dataset

## Dashboard Features
- KPI Cards
- Attrition Analysis
- Gender Distribution
- Age Distribution
- Interactive Slicers
- Business Insights

## Key Insights
- Research & Development department has highest attrition
- Most employees belong to the age group 25–35
- Male employees are higher in count
- Higher salary roles show lower attrition
